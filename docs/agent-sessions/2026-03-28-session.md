# 2026-03-28 Session - Initial MVP Build

## Goal
Build a desktop app (Tauri v2 + Svelte 5) as a frontend for ffmpeg, enabling easy file format conversion via drag & drop.

## Files Changed
- `src-tauri/src/lib.rs` - Tauri app setup with shell, dialog, opener plugins
- `src-tauri/src/commands.rs` - Rust command for ffmpeg sidecar invocation with progress tracking
- `src-tauri/tauri.conf.json` - App config with sidecar externalBin, window settings
- `src-tauri/Cargo.toml` - Added tauri-plugin-shell, tauri-plugin-dialog, regex
- `src-tauri/capabilities/default.json` - Shell spawn permission for ffmpeg sidecar
- `src-tauri/binaries/ffmpeg-x86_64-unknown-linux-gnu` - Bundled ffmpeg static binary (77MB)
- `src/app.css` - Tailwind CSS imports
- `src/routes/+page.svelte` - Main app page with full conversion flow
- `src/routes/+layout.svelte` - Layout wrapper for Tailwind
- `src/lib/types.ts` - TypeScript types (InputFile, ConversionStatus, FormatOption)
- `src/lib/utils/formats.ts` - Format definitions, media type detection
- `src/lib/stores/conversion.ts` - Svelte stores for app state
- `src/lib/components/DropZone.svelte` - Drag & drop / click to browse zone
- `src/lib/components/FormatPicker.svelte` - Output format dropdown
- `src/lib/components/ProgressBar.svelte` - Conversion progress bar
- `src/lib/components/ConvertButton.svelte` - Convert trigger button
- `tailwind.config.js` - Tailwind content paths
- `package.json` - Added @tauri-apps/plugin-shell, @tauri-apps/plugin-dialog, tailwindcss
- `.gitignore` - Added target/, binaries/ffmpeg-*
- `scripts/download-ffmpeg.sh` - Helper to download static ffmpeg builds

## Commands Run
- `npm create tauri-app@latest . --manager npm --template svelte-ts --yes --force --tauri-version 2`
- `npm install -D tailwindcss@3 postcss autoprefixer && npx tailwindcss init -p`
- `npm install @tauri-apps/plugin-shell @tauri-apps/plugin-dialog`
- `npm run build` (frontend only)
- `bash scripts/download-ffmpeg.sh linux`
- `npm run tauri build`

## Build Outputs
- Binary: `src-tauri/target/release/tauri-app` (16MB)
- DEB: `src-tauri/target/release/bundle/deb/ez-ffmpeg_0.1.0_amd64.deb` (34MB)
- AppImage: `src-tauri/target/release/bundle/appimage/ez-ffmpeg_0.1.0_amd64.AppImage` (125MB)

## Notes
- ffmpeg sidecar uses streaming (spawn) for real-time progress via stderr parsing
- Progress calculated from `Duration:` and `time=` fields in ffmpeg output
- Tauri drag/drop events (tauri://drag-enter, tauri://drag-leave, tauri://drag-drop) for OS file drops
- File dialog fallback for browsing
- To add Windows support: run `bash scripts/download-ffmpeg.sh windows`
